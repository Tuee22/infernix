# Infernix Legacy Tracking For Deletion

**Status**: Authoritative source
**Referenced by**: [README.md](README.md), [development_plan_standards.md](development_plan_standards.md)

> **Purpose**: Provide the explicit ledger of obsolete paths, duplicate guidance, and future
> cleanup work in `infernix`.

## Scope

- this ledger tracks implementation placeholders, compatibility shims, duplicate definitions, and
  stale guidance that still exists in the worktree or tracked index
- ordinary UI placeholder copy is not tracked here unless it preserves a fallback behavior or
  masks a live platform failure

> **Apple Silicon validation reset (2026-05-29).** The repository moved its primary development
> machine to a new Apple Silicon host on 2026-05-29, and the prior Apple Silicon and Linux/CUDA
> machines that produced the May 2026 validation dates cited below are no longer the active
> validation hosts. Every "validated on May 25-29, 2026" or "passed May 26-27, 2026" reference
> in the Pending Removal and Completed tables below reflects a proof point on the retired
> hardware. The underlying code changes are real and present in the worktree; the validation
> evidence must be reproduced on the current cohort hosts before any cleanup row that currently
> reads "validated" can be treated as a current proof point. Rows dated 2026-06-06 in this ledger
> record proof points reproduced on the current Apple Silicon host.

implicit topic-creation reliance, Apple engine singleton primary-lock replacement, single Apple

remaining pending row tracks the Webapp-role consolidation.
